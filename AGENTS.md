# AGENTS

## Project

Shuttle ETA — Real-time shuttle arrival predictions for UNILAG students.
Built for the PESSA Innovation Challenge 2026.

## Tech Stack

- React 19 + TanStack Start (SSR, file-based routing)
- Tailwind CSS 4 + shadcn/ui
- Vite 8
- TypeScript

## Commands

- `pnpm dev` — Start dev server
- `pnpm build` — Production build
- `pnpm lint` — ESLint
- `pnpm format` — Prettier
